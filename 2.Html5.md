### 結構樣板
```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body>
    <header>表頭</header>
    <nav>導覽</nav>
    <article>
        文章主體
        <section>文章內容部分(段落)</section>
    </article>
    <aside>側邊欄</aside>
    <footer>表尾</footer>
</body>

</html>
```
